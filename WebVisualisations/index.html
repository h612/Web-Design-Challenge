    Summary

    The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assesmble a dataset
    on over 500 cities.

    Visualiations MatplotLib

    After assembling the dataset, we used Matplotlop to plot various aspects of the weather v.s. latitude patterns.
    Factors we looked into included..Cloudiness, windspeed, and humidity.
